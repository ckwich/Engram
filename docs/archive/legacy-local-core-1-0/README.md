# Legacy Local-Core 1.0 Docs

These documents are archived because they describe the older local-core 1.0
finish line. They are retained for historical context only.

The active Engram 1.0 target is now the Memory OS rebuild described in:

- `docs/ENGRAM_MEMORY_OS_REBUILD_SPEC.md`

In the rebuild target, SQLite ledger, content-addressed source storage, LanceDB
retrieval, Kuzu graph storage, and daemon-owned runtime are part of Engram 1.0.
Hosted operation, sync, tenant auth, billing, and hosted collaboration are
post-1.0 work.
